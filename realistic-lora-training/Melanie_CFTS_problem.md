## Цель

Обучить персонажную LoRA, которая генерирует **2D-персонажа**, узнаваемо похожего на реального человека (Мелани Лоран, кадры из фильма).

- Стиль: иллюстративный / аниме (Illustrious), не фотореализм
- Сходство: форма лица, ключевые черты (родинки, глаза, губы), а не только атрибуты (блонд, костюм)
- Гибкость: работает на разных промптах и позах, не только «кадр из фильма»

**Триггер:** `Melanie_CFTS`  
**Датасет:** `D:\SD\datasets\Characters\Melanie`  
**Рабочая LoRA:** `D:\SD\lora_output\Melanie_CFTS_v2\` (база `illustriousXL_v01`)

---

## Текущие симптомы

| Симптом | Описание |
|---|---|
| Частичное сходство | v2 узнаваем в целом, но есть отличия от оригинала |
| Атрибуты ОК, identity слабо | Блонд, глаза, костюм — да; форма лица, родинки — нет |
| Привязка к сцене | Склонность воспроизводить позы/контекст из обучающих кадров |
| Нестабильность | Сходство зависит от промпта, веса LoRA, эпохи |

---

## Особенность задачи: photo → 2D

Датасет — **реальные кадры**, целевой рендер — **2D**. LoRA должна извлечь identity из фото и перенести на иллюстративный прайор базовой модели. Это сложнее, чем обучение на готовом 2D-арте персонажа.

---

## Датасет

52 изображения + 52 `.txt` капшена. Все кадры — скрины из одного видео (`Снимок плеера1…65`).

| Факт | Значение |
|---|---|
| Качество исходников | Достаточное: текстура кожи, родинки видны на референсах |
| Ракурсы | Почти только крупный/средний план; мало профилей, эмоций |
| Разрешение | Среднее ~842×783 px, `bucket_no_upscale: true` |
| Капшены | WD14, не редактировались; везде тег `realistic` |
| `brown hair` / `blonde hair` | 14 / ~47 файлов |
| Шумные теги (`blurry`, `blood`, `sleeping`…) | ~19 файлов |

Примеры: `Снимок плеера2.txt` — `brown hair, grey eyes, blurry`; `Снимок плеера10.txt` — `blonde hair, blood on face`.

## Базовая модель (2D-задача)

| Файл | Роль |
|---|---|
| `illustriousXL_v01.safetensors` | Текущая база v2; чистый 2D |
| `illustriousRealism_ilXL10V30.safetensors` | Полу-реалистичный 2D — кандидат, если v2 слишком «вайфу» |
| `perfectdeliberate_v10RL.safetensors` | Путь фотореализма (v4); для 2D-задачи не используется |
